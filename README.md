# DiagnoLearn - AI Diagnostic Learning Engine

Upload a question paper and answer sheet. Get a personalised weakness report and study plan in under 10 seconds.

## Tech Stack
- Python + Flask
- Gemini Vision AI / Groq Llama 3
- HTML, CSS, JavaScript

## Setup
1. Clone the repo
2. pip install flask google-genai python-dotenv pypdf
3. Create .env file with your GEMINI_API_KEY
4. python app.py
5. Open http://localhost:5000
